# numeric-integration

Pure-Rust numerical integration methods with no external dependencies.

## Features

- **Trapezoidal rule** — composite trapezoidal integration
- **Simpson's rule** — 1/3 rule, exact for cubic polynomials
- **Romberg integration** — Richardson extrapolation for high accuracy
- **Gaussian quadrature** — Gauss-Legendre nodes (1–5 points)
- **Adaptive quadrature** — Adaptive Simpson's with error estimation
